# CedarNexus Technologies - Marketing Website

A professional B2B marketing website for CedarNexus Technologies, built with Next.js 14, TypeScript, and Tailwind CSS.

## Overview

CedarNexus Technologies builds AI agents and automation solutions for service businesses like HVAC companies and dental practices. This website showcases our services, solutions, and industry expertise.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **React** - UI component library

## Features
- ✅ Modern, professional B2B design
- ✅ Fully responsive and mobile-friendly
- ✅ SEO optimized
- ✅ Accessible navigation with dropdown menus
- ✅ Multiple content pages
- ✅ Contact form
- ✅ Clean, minimal aesthetic with strong typography

## Pages

- **Home** (`/`) - Hero section, key capabilities, industries served
- **Services** (`/services`) - Detailed service offerings
- **Solutions** (`/solutions`) - Complete AI solutions by category
- **Industries** (`/industries`) - HVAC & home services, dental & healthcare
- **What we think** (`/what-we-think`) - Insights and thought leadership
- **Careers** (`/careers`) - Open positions and company culture
- **Contact** (`/contact`) - Contact form and information

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build

Create a production build:

```bash
npm run build
```

### Start Production Server

```bash
npm start
```
